# プロジェクト概要: Radio Calisthenics Together

毎朝、自動でラジオ体操のYouTube Live配信を行うためのシステムです。

## Phase 1 開発目標
- 毎朝指定時刻にYouTube Live配信を自動で開始できる。
- OBSを自動起動し、指定シーンで配信開始できる。
- 指定時間（例：15分）で自動停止できる。
- すべてMacローカルで完結（追加サーバ不要）。
- ログにより、失敗時の原因追跡が可能。

## システムアーキテクチャ
- **制御ロジック**: Python (obsws-python)
- **配信エンジン**: OBS Studio
- **スケジューラ**: macOS launchd
- **設定管理**: .env ファイル

## ディレクトリ構成
- `docs/`: ドキュメント一式
- `config/`: 設定ファイル（launchd用、OBS設定メモ等）
- `scripts/`: 実行用スクリプト（開始、停止、状態確認、インストール）
- `src/rct/`: 共通ロジック（OBS操作、ログ出力、設定読み込み）
- `logs/`: 実行ログ保存用
- `tests/`: 最小限のテストコード
